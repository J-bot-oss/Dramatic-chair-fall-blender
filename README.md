# Dramatic Chair Fall Animation

This is a beginner-friendly Blender animation project where a simple chair overreacts dramatically after seeing a tiny ball roll toward it.

## Project Idea

The animation is a short silly scene designed for platforms like TikTok and YouTube Shorts. A small ball rolls toward a chair, the chair reacts in fear, shakes, slides away, and falls dramatically.

## Tools Used

- Blender
- Basic mesh objects
- Keyframe animation
- Eevee rendering
- Vertical video format for short-form content

## What I Practiced

- Creating simple objects using basic shapes
- Building a chair from cubes
- Setting up a basic room environment
- Adding materials and lighting
- Animating objects with keyframes
- Camera positioning for vertical video
- Rendering frames/video for social media

## Animation Breakdown

1. The ball starts off-screen/away from the chair.
2. The ball rolls toward the chair.
3. The chair notices the ball.
4. The chair shakes dramatically.
5. The chair slides away in fear.
6. The chair falls over.
7. The scene ends with the chair on the floor.
